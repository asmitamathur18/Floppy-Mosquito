# Floppy Mosquito

Floppy Mosquito is a simple browser-based arcade game inspired by classic one-button games. The player controls a mosquito and navigates it through moving obstacles using tap or click inputs.

The project is built to demonstrate core front-end concepts such as animation loops, collision detection, event handling, and basic game logic using vanilla web technologies.

- Real-time collision detection
- Score tracking
- Responsive browser support
- Lightweight and fast load time
